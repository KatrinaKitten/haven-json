com.tripl3dogdare.havenjson.JsonNamePolicy

Sets the desired NamePolicy for a given JsonSchema subclass. If this annotation is not present, NamePolicy.AsWritten will be assumed. If multiple NamePolicys are passed, they will be applied from left to right. If NamePolicy.Custom is passed, the JsonSchema subclass must define a companion object inheriting from CustomNamePolicy.
