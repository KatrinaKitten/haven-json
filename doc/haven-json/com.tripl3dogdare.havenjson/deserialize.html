fun <T : JsonSchema> JsonValue.Companion.deserialize(constructor: KFunction<T>, raw: Json, nameConverter: NameConverter = JsonSchema.AS_WRITTEN, deserializers: Deserializers = Deserializers): T

Deserialize a JsonValue via the given constructor.

Parameters

T - The type to return. Must inherit from JsonSchema.

constructor - The constructor function to call.

raw - The raw JsonValue.

nameConverter - The function to use to convert JSON property names into matching constructor parameter names.

deserializers - The Deserializers instance to use for custom deserialization functions.

Exceptions

NullPointerException - if a non-nullable parameter corresponds to a JSON property containing null.

ClassCastException - if the passed JsonValue is not a JsonObject.

ClassCastException - if a parameter corresponds to a JSON property of the wrong type.

NoSuchFieldException - if a JSON property is found without a corresponding constructor parameter.

Return
The results of calling the given constructor with named parameters corresponding to the property names of the given JSON object.
